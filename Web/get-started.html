Agent Handoff

Paste once. Let the agent handle login.

This page is not an install wizard. It is a handoff document for OpenClaw-style tools. Copy the Markdown, paste it into your agent, and let it install Cookey, start a login request, and wait for the browser session to arrive.

1. Copy the doc Grab one Markdown block, not a pile of tabs and snippets.
2. Paste to agent Use OpenClaw, OpenHands, Goose, or another terminal agent.
3. Scan and finish Your iPhone completes login, then the session lands in the CLI.
See What It Contains

The copied text includes the install action, the run sequence, and the exact command shape the agent should follow.
