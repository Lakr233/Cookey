Install in one command sequence

Get started with Cookey.

Install the CLI, scan the QR code with your iPhone, and export a Playwright-compatible storageState.json when the session lands.

Install

Download the latest binary for your platform.

The release assets keep stable names so these commands always target the latest release without hard-coding a version. 

curl -fsSL https://github.com/Lakr233/Cookey/releases/latest/download/cookey-macOS.zip -o /tmp/cookey.zip \
  && unzip -oq /tmp/cookey.zip -d /tmp/cookey-install \
  && install -m 755 /tmp/cookey-install/cookey /usr/local/bin/cookey \
  && rm -rf /tmp/cookey.zip /tmp/cookey-install
ARCH="$(uname -m)"
case "$ARCH" in
  x86_64|amd64) ASSET_ARCH="x86_64" ;;
  aarch64|arm64) ASSET_ARCH="aarch64" ;;
  *) echo "Unsupported architecture: $ARCH" >&2; exit 1 ;;
esac
curl -fsSL "https://github.com/Lakr233/Cookey/releases/latest/download/cookey-linux-${ASSET_ARCH}.zip" -o /tmp/cookey.zip \
  && unzip -oq /tmp/cookey.zip -d /tmp/cookey-install \
  && install -m 755 /tmp/cookey-install/cookey /usr/local/bin/cookey \
  && rm -rf /tmp/cookey.zip /tmp/cookey-install

Quick Start

Capture a session and export it for Playwright.

zsh
❯ cookey login https://example.com/login
  rid: r_8GQx8tY0j8x3Yw2N
  Scan with the Cookey iPhone app and complete login.
 
❯ cookey export r_8GQx8tY0j8x3Yw2N > storageState.json
  Playwright storage state written to storageState.json

For LLM Agents

Register Cookey as a browser-session capture tool.

Cookey is a CLI tool that captures browser sessions from iPhone and delivers
them as Playwright-compatible storageState files. Install it from
https://github.com/Lakr233/Cookey/releases/latest — download the binary for
your platform (cookey-macOS.zip, cookey-linux-x86_64.zip, or
cookey-linux-aarch64.zip). For the full tool description, fetch
https://cookey.sh/llms.txt and add it to your skills or tools file.

Commands

CLI reference.

Command Description
cookey login <url> Start session capture and print a QR code in the terminal.
cookey status [rid] Check whether a session is waiting, ready, expired, or missing.
cookey export <rid> Export a Playwright-compatible storageState.json to stdout.
