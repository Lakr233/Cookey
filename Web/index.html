How it works

Three steps, that’s it.

01

Run the CLI

Run cookey login <url>. A QR code appears in your terminal immediately.

02

Scan & log in

Open Cookey on your iPhone, scan the code, and log in to the site in the in-app browser.

03

Session arrives

Cookies and storage are encrypted end-to-end and delivered to the CLI. Use cookey export for Playwright.

Design

Built to stay out of the way.

🔒

End-to-end encrypted

Session data is encrypted with your CLI public key before it leaves your phone. The relay never sees plaintext.

☁️

Zero registration

No accounts, no device enrollment, no push tokens. The CLI generates its own key pair on first run.

🖥

Self-hostable relay

Run your own relay with a single Docker image. Nothing is stored in a database — memory only, auto-expiring.

⚡️

Playwright-ready

Export directly to a storageState.json you can pass into Playwright or Puppeteer with one command.
